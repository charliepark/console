# Tabs

The `Tabs` component is just a CSS skin on [@reach/tabs](https://reach.tech/tabs/), which handles accessibility features and provides a very clean API.

## Full width

To get full-width tabs, simply use `flex-1` on the `Tab`s. `TabList` has `display: flex` [by default](https://github.com/reach/reach-ui/blob/d2864c99f497f69ba0dd4a6d980209285945b46b/packages/tabs/styles.css#L10).
